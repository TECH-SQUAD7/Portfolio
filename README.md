# Rajat Portfolio

Anime-inspired personal portfolio built with Next.js, Tailwind CSS, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy

This repo includes a GitHub Actions workflow for GitHub Pages. After pushing to `main`, enable Pages with GitHub Actions as the source if GitHub asks for it.

## Notes

- The music card/track section is intentionally removed.
- The visual system uses original CSS-built sky, glass, cloud, paper-plane, and silhouette elements instead of copyrighted anime artwork.
- Replace placeholder contact links and add images under `public/images/` when needed.
